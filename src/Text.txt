## useState() && form & Controlled input 

    useState is a react  hook that allows you to add state (data that can change over time)
    to a functional component.

    normally ,in js functionas, variable disappear after the function runs.
    but with useState, react remember value between re-render.

    useState is a React Hook that allows functional components to store 
    and manage data (state) and re-render the UI when that data changes.

    syntax:

        const[state,setState]=useState(initialvalue)

        state-- the curent value of the variable.
        setState-- a function that updates the  state.
        initialvalue -- the starting  value of  the state.
